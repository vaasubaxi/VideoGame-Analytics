# Video Game Analytics
### Project Overview
This project explores a dataset of video games sourced from Backloggd via Kaggle, containing information on game titles, genres, release years, platform availability, user ratings, and engagement metrics including wishlists, backlogs, and active players.

The central question driving this analysis is: "Game developers spend millions building hype — but are players actually playing the games they wishlist?"

The analysis investigates the disconnect between player anticipation and actual engagement. Metrics like wishlists and reviews reflect pre-release hype, while playing and backlog counts reveal what happens after launch. Understanding this gap has real implications for how publishers forecast a game's success.
